# 75 Soft Tracker

A static GitHub Pages web app for tracking a 75 Soft health challenge without a gym membership.

## Features

- 75-day challenge progress
- Daily checklist
- No-gym activity picker
- 3-liter water tracker
- Daily notes
- Streak and completion stats
- Clickable 75-day history
- Local browser storage
- Installable PWA support

## Publish with GitHub Pages

1. Open the repository settings.
2. Go to **Pages**.
3. Set source to **Deploy from a branch**.
4. Choose **main** and **/root**.
5. Save.

The app should publish at:

`https://alexthezero.github.io/75-soft-tracker/`
